# Forum site

!*Without mobile adaptation*

### Site on python *( django )*

**Install and run virtual environment**
> pipenv shell

**Install dependencies**
> pipenv install

## Tools used 
- bootstrap
- Mysql
- django `4.1.3`
    - ckeditor
    - django-simple-captcha
    - django-debug-toolbar
    - django-rest-framework
- Celery
- Redis 
- Javascript
    - jquery
